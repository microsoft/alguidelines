## Description
A line of code should not have more than one statement.

## Bad code

```al
    if OppEntry.Find('-') then exit;  
```   
    

## Good code

```al
    if OppEntry.Find('-') then   
        exit;  
```   
    
## Bad code

```al
    TotalCost += Cost; TotalAmt += Amt;  
```   
    

## Good code

```al
    TotalCost += Cost; 
    TotalAmt += Amt;
```
