## Description

Always start comments with // followed by one space character.

## <span style="color:red">Bad code</span>

```al
RowNo += 1000; //Move way below the budget  
```

## <span style="color:lime">Good code</span>

```al
RowNo += 1000; // Move way below the budget
```
