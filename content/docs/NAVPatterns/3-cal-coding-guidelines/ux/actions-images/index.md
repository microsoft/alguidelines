All actions must have an image assigned to them.

Bad code

```al
{ 7 ;1 ;Action ;
        CaptionML=ENU=Customer - &Balance;
        RunObject=Report 121 }
```

Good code

```al
{ 7 ;1 ;Action ;
        CaptionML=ENU=Customer - &Balance;
        RunObject=Report 121 }
        Image=Report }
```
